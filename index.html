Preguntas Frecuentes

Tengo más de 5 años de experiencia en desarrollo web, trabajando con diversas tecnologías como HTML, CSS, JavaScript, React, Node.js, y bases de datos como MongoDB y Firebase.

Puedes contactarme a través del formulario en la sección de Contacto o enviarme un correo electrónico a [tu correo electrónico].

Utilizo una combinación de herramientas según las necesidades del proyecto. Para diseño visual, empleo Figma o Adobe XD. En el desarrollo, trabajo con HTML, CSS y JavaScript para el front-end, mientras que para el back-end suelo usar Node.js o Firebase. Además, utilizo Git y GitHub para el control de versiones y la colaboración en equipo.

Me aseguro de que todos los sitios web que desarrollo sean accesibles siguiendo las pautas de accesibilidad WCAG. Esto incluye el uso de etiquetas semánticas en HTML, textos alternativos en imágenes y un diseño que funcione bien con lectores de pantalla. Para la responsividad, utilizo técnicas como media queries en CSS y frameworks como Bootstrap para garantizar que el sitio se adapte a cualquier dispositivo o resolución.
